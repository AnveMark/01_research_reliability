## üîç What Reliability Really Means ‚Äî Global Research Standards

**ESOMAR (European Society for Opinion and Marketing Research)** defines reliability as:

> ‚ÄúThe degree to which a measurement system produces stable and consistent results over time and across researchers, respondents, and environments.‚Äù

In statistical form:

The goal of reliability ‚â† **eliminating all error**.  
The goal = **reducing avoidable error so insights remain stable across replications**.

## ‚ö†Ô∏è 2025 ESOMAR Guidelines: Reliability MUST Comply With These 6 Principles

| ESOMAR Guideline Principle    | Relevance to Reliability                                                                     |
| ----------------------------- | -------------------------------------------------------------------------------------------- |
| **Transparency**              | Instruments, sample, procedures, and biases must be openly documented                        |
| **Replicability**             | Research must be designed so another researcher can re-run the study and get similar results |
| **Standardisation**           | Protocols must be consistent across geographies, researchers & modes                         |
| **Data Integrity**            | Avoid distorted measurement ‚Äî neutral questions & realistic stimulus                           |
| **Respondent Protection**     | No pressure or inducement to give socially desirable answers                                   |
| **Disclosure of Limitations** | Reports must state reliability risks & sources of uncertainty                                  |

üìå If research is **not replicable ‚Üí it does not meet 2025 ESOMAR reliability standards.**

## üß† Where Reliability Breaks - 12 Real-World Sources of Error

| Category                 | Example Result                                        |
| ------------------------ | ----------------------------------------------------- |
| Measurement Bias         | Suggestive wording ‚Üí inflated satisfaction            |
| Sampling Bias            | Convenience sampling ‚Üí false demand signals           |
| Temporal Bias            | Festive-period tests ‚Üí false baseline volumes         |
| Stimulus Bias            | Unrealistic pack rendering ‚Üí inflated purchase intent |
| Mode Bias                | CATI vs CAWI differences ‚Üí variation in responses     |
| Interviewer Bias         | Tone changes ‚Üí rating changes                          |
| Order Effect             | Listing brand first ‚Üí higher preference               |
| Recency Bias             | Last-viewed ad remembered ‚Üí false recall              |
| Social Desirability Bias | Parenting/hygiene categories ‚Üí flattering answers     |
| Satisficing Bias         | Straight-lining & fast completes                      |
| Anchoring Bias           | Benchmarks shared upfront ‚Üí converged scores          |
| Halo Effect              | Liking of one feature ‚Üí inflated overall rating       |

> Bias management is **core to research design**, not just post-analysis cleanup.
